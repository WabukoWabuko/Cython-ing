Lesson 3 — Static Typing & Speed

Learn how adding C types in Cython impacts performance and how to identify hotspots.

Example (Python vs typed Cython)
# Python
 def sum_loop(n):
     total = 0
     for i in range(n):
         total += i
     return total

# Typed Cython (concept)
# cdef long sum_loop(long n):
#     cdef long total = 0
#     cdef long i
#     for i in range(n):
#         total += i
#     return total

Use `cython -a` to inspect C generated and find Python interactions.

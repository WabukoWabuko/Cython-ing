Lesson 2 — Compiling Your First Module

This lesson walks through creating a `.pyx` file, a `setup.py`, and building the extension locally.

Local lab steps
  1. Create `example.pyx` with a small function.
  2. Create `setup.py` to build the extension.
  3. Run `python setup.py build_ext --inplace` and import the module.
Example `.pyx`
cdef public int add_ints(int a, int b):
    return a + b
Example `setup.py`
from setuptools import setup
from Cython.Build import cythonize

setup(
    ext_modules=cythonize("example.pyx")
)

See the workshop pack for a full example repo and automated test that builds the extension in CI.

Lesson 4 — Integrating C Libraries

How to `cdef extern from` C headers and wrap a small C function for Python use.

Example
// small.c
int triple(int x){ return x * 3; }

// wrapper.pyx
cdef extern from "small.h":
    int triple(int x)

cpdef int triple_py(int x):
    return triple(x)

Workshop pack contains a full example repo with CI that builds the extension and runs tests.
